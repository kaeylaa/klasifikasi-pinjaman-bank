A. Overview  

Project ini bertujuan untuk memprediksi kemungkinan **default pinjaman nasabah bank** menggunakan beberapa metode machine learning.  
Pendekatan yang digunakan berfokus pada **perbandingan performa model klasifikasi**, sehingga dapat diketahui model mana yang paling optimal dalam kasus ini.

---

B. Dataset  

Dataset berisi informasi karakteristik nasabah, meliputi:  

- Age  
- Education  
- Employment  
- Address  
- Income  
- Debt-to-Income Ratio  
- Credit Debt  
- Other Debt  

Target:  
- **Default (0 = No, 1 = Yes)**  

---

C. Models  

Model yang digunakan dalam penelitian ini:  
- Artificial Neural Network (ANN)  
- Support Vector Machine (SVM)  
- Decision Tree  
- Random Forest  

---

D. Methodology  

Tahapan analisis yang dilakukan:  

1. Data Cleaning  
2. Exploratory Data Analysis  
3. Data Splitting (80:20)  
4. Model Training  
5. Model Validation  
6. Model Evaluation  
7. Model Comparison  

---

E. Evaluation Metrics  

Performa model diukur menggunakan:  

- Accuracy  
- Precision  
- Recall  
- F1-Score  

---

F. Insight  

- Random Forest menunjukkan performa paling stabil  
- SVM optimal setelah tuning parameter  
- Decision Tree mudah diinterpretasi namun cenderung overfitting  
- ANN mampu menangkap pola kompleks
